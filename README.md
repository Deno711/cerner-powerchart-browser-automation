## 📋 About cerner-powerchart-browser-automation

This application helps you use Cerner PowerChart faster and easier. It does this by automating tasks in your web browser. With it, you can extract clinical data, manage orders, automate documentation, and move through PowerChart without needing to use any APIs. You don’t need technical skills to use this tool. It works in the background while you navigate your health records.

The automation works with tools called Playwright and Puppeteer behind the scenes. These tools control the browser to save you time and reduce repeated work.

**Can I use this without logging into PowerChart?**

No. The app requires an active PowerChart session to work.

**Is this safe to use with my patient data?**

The app only automates your current browser session. Your data never leaves your device unless you choose to share saved files.

**What if the automation stops working?**

Update the app from the download page to get the latest fixes. Restart your computer if needed.

**Can I cancel an automation mid-task?**

Yes. Use the stop button in the app to cancel any running process.
